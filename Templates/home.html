Abstract


Identifying a brain tumour by hand in the present days is a very time-consuming and imprecise task due to the vast amount of MRI (magnetic resonance imaging) pictures that must be checked. It might compromise a patient's ability to receive the best possible care. Once more, because it contains a lot of steps, it can take a long time.several image datasets in a large quantity. Both normal tissue and brain tumours share a lot of similarities. so there is automatic tumour detection technique in which CNN model will be used which help to detcet the brain tumor in mri images. In which convolutional neural networks (CNN) can handle the image type data.which help to increase speed in treatment.The Technique used here is OTSU ThresholdingWhich changes the Image into Grey-format.This Technique we are using follows semantic segmentation.
Keywords:CNN Model,Semantic Segmentation,OTSU Thresholding
